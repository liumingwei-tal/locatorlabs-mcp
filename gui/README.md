# LocatorLabs 浏览器客户端

一个基于 Electron 的桌面浏览器客户端，可以在应用内打开和浏览网页，支持多标签页。

## 功能特性

- 🌐 在应用内浏览网页（所有页面都在客户端内打开）
- 📑 **多标签页支持** - 可以同时打开多个网页
- ➕ 新建标签页按钮
- 🔒 自动拦截外部链接，确保在客户端内打开
- 🔍 智能URL格式化（自动添加https://）
- ⬅️➡️ 前进/后退导航
- 🔄 页面刷新
- 📊 实时加载状态显示（每个标签页独立显示）
- ⚠️ 错误提示
- 🎨 现代化标签页界面

## 安装依赖

首先确保已安装所有依赖：

```bash
npm install
```

## 启动GUI客户端

```bash
npm run gui
```

或者：

```bash
electron gui/main.js
```

## 使用方法

1. **输入URL**: 在顶部地址栏输入网址
   - 支持完整URL（如 `https://www.example.com`）
   - 支持域名（如 `example.com`，会自动添加 `https://`）
   - 支持搜索查询（会使用百度搜索）

2. **标签页操作**:
   - 点击"+"按钮创建新标签页
   - 点击标签页切换不同页面
   - 点击标签页上的"×"关闭标签页
   - 所有外部链接会自动在新标签页中打开

3. **导航**:
   - 点击"前往"按钮或按回车键在当前标签页加载页面
   - 使用后退/前进按钮浏览当前标签页的历史
   - 点击刷新按钮重新加载当前标签页

4. **查看状态**: 底部状态栏显示当前活动标签页的加载状态和标题

## 界面说明

- **工具栏**: 包含Logo、URL输入框和导航按钮
- **标签页栏**: 显示所有打开的标签页，支持自适应高度
- **浏览器视图**: 显示网页内容，支持完整显示
- **状态栏**: 显示加载状态和页面标题

## 技术栈

- Electron - 跨平台桌面应用框架
- WebView - 内嵌浏览器组件

## 系统要求

- Node.js 18+
- Electron 28+
